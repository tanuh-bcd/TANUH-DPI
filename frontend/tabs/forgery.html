 Usage Examples




    

        
1. Request a token

        
Returns an access_token valid for 24 hours.

        

            
curl -X POST "https://dpi-dev.tanuh.ai/forgensic/api/token" \
  -H "Content-Type: application/json" \
  -d '{"name": "Your Name", "email": "you@example.com"}'

            
        

    

    

        
2. Upload a document

        
Send as multipart/form-data. Returns a job_id.

        

            
curl -X POST "https://dpi-dev.tanuh.ai/forgensic/jobs" \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@/path/to/document.pdf" \
  -F "ocr_enabled=true"

            
        

    

    

        
3. Poll job status

        
Poll until status is "complete" or "error".

        

            
curl -H "Authorization: Bearer $TOKEN" \
  "https://dpi-dev.tanuh.ai/forgensic/jobs/JOB_ID"

# Response:
{
  "job_id": "abc123",
  "status": "processing",
  "progress": 0.42
}

            
        

    

    

        
4. Fetch results + preview

        
Results include preview URLs; fetch them with the same token.

        

            
curl -H "Authorization: Bearer $TOKEN" \
  "https://dpi-dev.tanuh.ai/forgensic/jobs/JOB_ID/results"

curl -L -H "Authorization: Bearer $TOKEN" \
  "https://dpi-dev.tanuh.ai/forgensic/jobs/JOB_ID/files/preview_document_page_1.jpg" \
  -o preview.png

            
        

    







    
Python: End-to-end flow

    

        
import time, requests

BASE = "https://dpi-dev.tanuh.ai/forgensic"

token = requests.post(
    f"{BASE}/api/token",
    json={"name": "Your Name", "email": "you@example.com"},
    timeout=20,
).json()["access_token"]

headers = {"Authorization": f"Bearer {token}"}

with open("document.pdf", "rb") as f:
    r = requests.post(
        f"{BASE}/jobs",
        files={"file": f},
        data={"ocr_enabled": "true"},
        headers=headers,
        timeout=60,
    )
job_id = r.json()["job_id"]

for _ in range(30):
    s = requests.get(f"{BASE}/jobs/{job_id}", headers=headers, timeout=30).json()
    if s["status"] in ("complete", "error"):
        break
    time.sleep(2)

results = requests.get(f"{BASE}/jobs/{job_id}/results", headers=headers, timeout=30).json()
print(results)

# Example: download the first preview image
first_page = results["pages"][0]
preview_url = first_page.get("preview_url") or first_page.get("image_url")
if preview_url:
    img = requests.get(f"{BASE}{preview_url}", headers=headers)
    open("preview.jpg", "wb").write(img.content)

 Privacy & Data Handling

    

        Uploads are written to a temporary job folder while processing and then cleaned up. Annotated preview files are cached in memory for the job and cleared on server restart. Jobs are also purged after the TTL window configured on the server.

Detection Pipeline

            
The Forgensic system runs multiple classical computer vision analyses on each page of the document. Each check produces localized findings with bounding-box coordinates.


            

                
1. Document Ingestion
PDF pages converted to images at high resolution. Direct image uploads used as-is.

                
2. Copy-Paste Detection
Duplicate region matching using template matching to find moved or duplicated content within the same document.

                
3. Overwrite Detection
Edge density and stroke inconsistency analysis to identify text or content written over existing marks.

                
4. Added Content Detection
Stamp and signature heuristics with color and shape cues to detect pasted or overlaid elements.

                
5. Erasure Detection
Low-noise gap analysis and unnatural smoothness detection in text lines indicating removed content.

                
6. Document Merge Detection
Noise fingerprint analysis across horizontal bands to detect pages assembled from different source documents.

                
7. Watermark & Spacing Analysis
Residual frequency artifacts from watermark removal, plus spacing anomalies in lines and token gaps.

                
8. Result Assembly
All findings aggregated with categories, confidence scores, and annotated preview images generated.

            

        


        
        

            
Finding Categories

            
Each detection finding is classified into one of the following categories:


            

                
IDLabelDescription

                
                    
C1Copy-pasteDuplicate regions within the same document

                    
C2OverwritingEdits over existing text or ink

                    
C3Added contentStamps, signatures, or pasted content

                    
C4ErasureRemoved text or image regions

                    
C5Document mergeHeader/body from different source documents

                    
C6Watermark removalGhost artifacts from watermark edits

                    
C7Irregular spacingSpacing inconsistencies in lines or tokens

                    
C8AI-generatedEntire document appears AI-generated

                    
C9AI-editedPartial AI edits detected in regions

                    
C10No discrepancyNo editing or anomalies found

Privacy & Data Handling

            
The Forgensic system is designed for secure, non-persistent document analysis.


            
    
                
  • No external calls — all processing happens locally using OpenCV, no cloud APIs
    • 
                
  • Temporary storage — uploaded files deleted immediately after processing
    • 
                
  • In-memory previews — annotated preview images cached in-memory only, cleared on restart
    • 
                
  • No persistence — no document content is written to disk or database
    • 
                
  • Job TTL — jobs automatically purged after configurable TTL window
    •

Python Example

            
import time, requests

BASE = "https://dpi-dev.tanuh.ai/forgensic"

token = requests.post(
    f"{BASE}/api/token",
    json={"name": "Your Name", "email": "you@example.com"},
).json()["access_token"]

headers = {"Authorization": f"Bearer {token}"}

with open("document.pdf", "rb") as f:
    r = requests.post(f"{BASE}/jobs", files={"file": f},
                      data={"ocr_enabled": "true"}, headers=headers)
job_id = r.json()["job_id"]

for _ in range(30):
    s = requests.get(f"{BASE}/jobs/{job_id}", headers=headers).json()
    if s["status"] in ("complete", "error"):
        break
    time.sleep(2)

results = requests.get(
    f"{BASE}/jobs/{job_id}/results", headers=headers
).json()
print(results["category_summary"])
